Court&Match User Guide

■ Overview

This app automatically generates match pairings and supports fair game management.

■ Basic Usage

  1. Add members
  2. Set the number of courts
  3. Start the match
  4. Press "Next" to proceed to the next match

■ Court Names

You can assign names to each court. If not set, default names like "Court X" will be used.

■ Auto Save

The app automatically saves the current state. You can resume your session after restarting.

■ Restore Behavior

  • Short time: Automatically restored
  • After some time: Confirmation dialog will appear
  • After long time: Automatically reset

■ Courts May Be Reduced

To ensure fairness in match and rest counts for all players, the number of active courts may be temporarily reduced depending on the situation.
