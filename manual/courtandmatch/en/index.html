Court&Match User Guide

■ Overview

This app automatically generates match pairings and supports fair game management.

■ Basic Usage

・Member Registration

You can register members as individuals (1 person) or pairs (2 people).
To register a pair, enable the "Pair" option.
This is intended for fixed pairs.

Example:
Player A (single)
Player B & Player C (pair)

・Member List

Registered members can be saved as a list.
Saved lists can be loaded later for quick setup.

After registering members, save them as a list.
You can load a saved list to quickly restore all members.

*Data is stored locally on the device.

・Court Settings

You can add courts using the "+" button.
To remove a court, press the "-" button.
*At least one court is required and cannot be removed.

You can set the match type for each court (Singles / Doubles / Triples).
Each type requires a specific number of players.
If there are not enough players, matches cannot be started.

・Start Match

Press "Start" to generate match pairings.

・During Match

Press "Next" to move to the next match.
You can adjust courts and members during matches.
If there are not enough players, adjust accordingly.

■ Court Names

You can assign names to courts.
If not set, default names like "Court X" will be used.

■ Auto Save

The app automatically saves the current state.
You can resume after restarting the app.

■ Restore Behavior

  • Short time: Automatically restored
  • After some time: Confirmation dialog appears
  • After long time: Automatically reset

■ Courts May Be Reduced

To ensure fairness in match and rest counts, the number of active courts may be temporarily reduced.
